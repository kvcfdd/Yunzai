# Commands

命令前缀：`#` = 原神, `#星铁` = 星穹铁道, `#绝区零` = 绝区零。
占位符说明：`{p}` = 前缀占位, `<x>` = 必填, `[x]` = 可选。

## 游戏数据

| 命令格式                | 命令说明                                                                                          | 示例                                                         |
| ----------------------- | ------------------------------------------------------------------------------------------------- | ------------------------------------------------------------ |
| `{p}绑定<uid>`          | 绑定对应游戏UID                                                                                   | `#绑定123456789` `#星铁绑定987654321` `#绝区零绑定987654321` |
| `{p}更新面板[uid]`      | 刷新面板数据，uid可选，默认已绑定                                                                 | `#更新面板` `#星铁更新面板` `#更新面板123456789`             |
| `{p}<角色名>面板[uid]`  | 查询指定角色面板数据                                                                              | `#胡桃面板` `#星铁三月七面板` `#绝区零安比面板`              |
| `#[x年x月]幻想角色列表` | 原神专属，查询指定时间的幻想真镜剧诗挑战信息和已绑定原神uid可参与本次挑战的角色列表信息，默认本月 | `#幻想角色列表` `#26年5月幻想角色列表`                       |
| `#<x年x月>深渊`         | 原神专属，查询指定时间的深境螺旋12层挑战信息                                                      | `#26年5月深渊` `#5月深渊`                                    |
